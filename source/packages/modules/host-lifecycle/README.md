# Host Lifecycle Module

Modulo responsavel por integracoes persistentes com o proprio PC.

## Escopo

- arranque automatico
- reparacao de integracao local
- coordenacao do companion de host
- ownership do ficheiro OAuth live usado pelo Codex

## Nota

Se o core app correr em `LXD`, este modulo deve ser carregado no `lume-hub-host`.
