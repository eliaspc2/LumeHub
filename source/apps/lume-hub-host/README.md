# Host App

Deployable local do projeto para responsabilidades do proprio PC.

## Responsabilidades

- gerir persistencia de arranque no host
- gerir deep sleep / wake lock
- gerir o mesmo ficheiro OAuth live usado pelo Codex
- servir de companion quando o core app corre em `LXD`

## Regra

Este app continua a pertencer ao mesmo projeto.
Nao e um script solto nem uma ferramenta externa.
